Installation guide

Maven dependency

Add following parts to your pom.xml 


    <dependencies>
        ... list of other dependencies ..
        <dependency>
            <groupId>ua.net.kurpiak</groupId>
            <artifactId>crawler</artifactId>
            <version>..suitable version here or don't use this configuration to use latest version</version>
        </dependency>
        ...
    </dependencies>


    ...

    <repositories>
        ... list of other repositories ...
        <repository>
            <id>myMavenRepoRead</id>
            <url>https://mymavenrepo.com/repo/Vhcdy0A146kUy2skztWx/</url>
        </repository>
    </repositories>

That's all

base-crawler is maintained by borsch.
